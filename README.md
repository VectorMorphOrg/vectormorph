# VectorMorph

## Adaptive Execution Graph Proof-of-Work

VectorMorph is an open-source research project exploring the next generation of decentralized computation through Adaptive Execution Graph Proof-of-Work (AEG-PoW).

Our goal is to develop a deterministic, adaptive consensus algorithm capable of exercising multiple computational domains while remaining suitable for decentralized blockchain systems.

Current Status

🚧 Pre-Alpha Research

Current Focus

• AEG-PoW
• EvoGraph
• Reference Hasher
• Protocol Specification
